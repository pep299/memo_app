# メモアプリ
## 必要要件
- メモの登録機能
- 登録したメモは閲覧可能
- ソート機能：新着順、古い順
- 表示件数を10件,25件,50件が指定可能
- ページング

## 対応バージョン
- Python 3.7.3
- Django 2.2.2
- その他ライブラリは`requirements.txt`参照

## 対象ブラウザ
- firefox 66.0.5
- Chrome 74.0.3729.131(一部レイアウト崩れあり)

## 発展内容
- postgres or MySQL
- nginx
- docker
- Heroku or AWS or GCP
